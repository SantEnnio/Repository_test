# Git
Software per il versionamento e la condivisione di file

- Condividiamo versioni di una cartella su disco
    - locale: commit
    - remoto: push

# Client git:

# Glossario:
- Repository: cartella versionata
- Commit: stato delle modifiche ad una cartella:
    - add file
    - remove file
    - edit file
